Get the data

May 5, 2019

“Blessed be all Heaven for sending us an adventure that is good for something!” - Sancho

The data is free for use and publicly available on the ESS website. You have to register on the ESS website to access the data: [http://www.europeansocialsurvey.org/user/new]

The registration form has to be filled only once. The email address that you used in your registration will be required to access the ESS data.

Importing rounds and countries

The package “essurvey” makes it very easy for you to access ESS data directly from R. First, you have to load the package by typing the following line:

library(essurvey)

Then, you can import data for either a specific round or a specific country. For this tutorial, we will work with two datasets:

  1. ESS round 8 (2016) with all countries
ess_eight <- import_rounds(rounds = 8, ess_email = "_____@___.com")
  

ess_eight <- import_rounds(rounds = 8, ess_email = "danielcapistrano@gmail.com")

With this command, we told R to do two actions:

a) Import rounds from the ESS repository with the following arguments:

  • rounds = specifies which rounds will be imported. For more than one round, you can use c(6,7,8)
  • email = specifies the user importing the data. Type between quotes the email address that you used for registration.

b) Create a data table from the result of the importation with the name of “ess_eight”. The operator "<-" (read assign) is used to give a name (on the left side) to the output of the command that you specified on the right side.

After running the command, the console will show a progress bar “Dowloading ESS8” and, after completion, the environment pane will show an object called “ess_eight”.


  1. Ireland dataset for all rounds
ess_ireland <- import_all_cntrounds(country = "Ireland", ess_email = "_____@___.com")
  

ess_ireland <- import_all_cntrounds(country = "Ireland", ess_email = "danielcapistrano@gmail.com")

With this command, R will import all rounds for “Ireland” and name this new object as “ess_ireland”.

The essurvey package has several other features to import or download ESS data. To check all the available functions and options of the package, you can read the package documentation typing the following line:

help(essurvey)
