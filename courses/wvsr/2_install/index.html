Install R

R is a free software that is widely used for data manipulation and analysis.

The software has a command line interface which might discourage those who are not familiar with programming. Therefore, we will use it with RStudio, an integrated development environment that will facilitate our work.

Here are the steps to install both R and RStudio:

  1. Download R from the Irish mirror or any other source listed here. Open the downloaded file and follow the instructions for installation.

  2. After installing R, download RStudio installer from here https://www.rstudio.com/products/rstudio/download/ and follow the instructions for installation.

RStudio interface

The RStudio interface is divided in four panes:

  • Source is a notepad for you to write, edit and document your code. What is written here will not be evaluated until you pres “Ctrl + Enter” or click the “Run” button on the top of the pane;

  • Environment shows the datasets and objects that you have created or loaded;

  • Files/Viewer provides an easy access to your folders and files. The pane also shows some outputs in the “Viewer” tab.

  • Console is where most of the output will be shown and where you can also type your code to be run.

Installing useful packages

Apart from the basic R functions, there are thousands of packages with additional functions to help users. Two of them are particularly useful for our basic analysis: tidyverse for data manipulation and visualisation, and essurvey to easily access and import ESS datasets.

To install these packages, open RStudio, type the following line at the console prompt and then hit enter:

install.packages(c("tidyverse", "essurvey"))
